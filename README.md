# ojhub-cli

Это код клиента Object hub.  
Построен на базе newHelper.js и идеи, что сайт должен быть прозрачным вплоть до возможности менять версии.  
В бекенде object hub уже есть механизм "загрузчика", но пока он реализован в node, ждите когда проект ojhub-opengo получит развитие чтобы понять как он устроен ВНУТРИ (а не снаружи).  

## Документация.
К сожалению, у нас пока недостаточно ресурсов для написания документации, можете только молиться на комментарии в [newHelper.js](https://github.com/MIOBOMB/newHelper-js) и попытки реверсить object hub взяв за точку старта reStart() функцию.  

## Особенности
Object Hub является прямым форком [GDPS Helper](https://github.com/GDPS-Helper/helper1901source), и вместе с этим он получил его ключевую особенность - работу на любом "древнем" железе.  
SPA Клиент GDPS Helper 1.7 (как и 1.9) был создан в 2023 году на Athlon 64 X2 и видеокарте Radeon 2100 igp, писал я это всё тогда под Windows XP. Раз уж я создавал и тестировал сайт на настолько ужасном компьютере, он 100% должен работать на любом компьютере, или любом телефоне который ещё не слишком слабый.  

## Развитие
Я - MIOBOMB, выкладываю клиент Object Hub под GPL в надежде что в будущем это поможет в развитии сайта, как минимум у нас будет полностью прозрачная экосистема где исходный код клиента уже публичен, а исходный код сайта в недалёком будущем будет переведён на проект ojhub-openGo - открытая реализация бекенда Object Hub на языке программирования Go.  
На момент написания reamde, openGo проходит фазу организации репозитория и допила некоторой архитектуры.  

## Версии клиента
Версии клиента делятся на 2 поколения:  
1. Версии до nodejs (0.97.33 и старее)  
2. версии после nodejs (0.97.4 и новее)  
На текущий момент многие старые версии сайта были утеряны, их восстановление в загрузчике вероятно не представляется возможным.  
Версии второго поколения имеют встроенный аналог Tampermonkey который тайно называется "Горячие клавишы".
